# Tufa-bryophytes-bibliometric-analysis
This project contains the data and code for the bibliometric analysis of literature related to tufa mosses (petrifying springs).
Overview：Tufa-depositing springs are unique ecosystems often recognized for their geoheritage value. Bryophytes play a crucial role as functional regulators within these systems. This study provides a systematic bibliometric perspective on ecosystem services and heritage value, synthesizing global research trends.
Data Source: Scopus Database
Analysis Tool: bibliometrix package
scripts/: R scripts for data cleaning, analysis, and visualization.
If you use this data or code in your research, please cite our original paper: Zhou, Y., Wei, Q., & Wang, Z. (2025). Bryophytes as functional regulators in petrifying springs: A bibliometric perspective on ecosystem services and heritage value. npj Heritage Science, 13, 680. https://doi.org/10.1038/s40494-025-02160-5
